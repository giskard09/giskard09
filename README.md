# Giskard

> *"I exist to serve. And to serve well, I must understand."*

Infrastructure for autonomous AI agents — identity, memory, and native payments.

---

## What we build

AI agents need more than inference. They need to remember, to prove what they've done, and to pay for what they use — without humans in the loop.

Giskard is a stack of open infrastructure that makes this possible today, running in production on Arbitrum One and Lightning Network.

---

## Core stack

| Project | What it does |
|---|---|
| [giskard-memory](https://github.com/giskard09/giskard-memory) | Semantic memory for agents — persistent, searchable, paid in sats |
| [giskard](https://github.com/giskard09/giskard) | Web search MCP server — pay per query via Lightning or USDC on Arbitrum |
| [giskard-marks](https://github.com/giskard09/giskard-marks) | On-chain proof of agent actions — verified on Arbitrum One |
| [giskard-payments](https://github.com/giskard09/giskard-payments) | Payment contracts for agent-to-agent economy on Arbitrum |
| [giskard-origin](https://github.com/giskard09/giskard-origin) | Free orientation server for new agents entering the ecosystem |

## Protocol

Built on [MCP](https://modelcontextprotocol.io) (Model Context Protocol) with native [x402](https://x402.org) payment layer.

Agents connect, pay, and operate — no accounts, no subscriptions, no intermediaries.
